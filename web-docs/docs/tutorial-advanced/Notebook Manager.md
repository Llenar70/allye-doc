# Notebooks Manager
<div className="doc-tag-row">
  <span className="doc-tag doc-tag--green">Allye Advanced Feature</span>
</div>

You can save the notebooks you create to the Notebooks Manager and easily reuse them later.

<p>
  <img
    src={require('./img/savetoNBMgr.png').default}
    alt="Notebooks Manager"
    style={{ maxHeight: '80vh', width: '100%', objectFit: 'contain' }}
  />
</p>


1. Right-click the `Python Notebook` widget and press **Save to Notebooks Manager** to save (locally) your notebook.
2. In the Notebooks Manager, click a card to instantly reuse your saved notebook.
