# Save 

## Canvas
1. Press cmd+s anywhere on the canvas or select `Save` from `File` in the menu bar to save the workflow as an `ows` file.
2. To restore the canvas from a saved `ows` file, drag and drop the `ows` file onto the canvas.

## Diagram Screen
1. To capture the workflow pipeline, use the OS screenshot (`cmd + shift + 4`) or select `Save Workflow Image as SVG` from `File` in the menu bar.
2. The saved file is an image and cannot be used to recreate the workflow on the canvas.

## Report
1. Click the report icon at the bottom left of a widget to open the Report Widget.
2. Use the `Save` button to choose HTML, PDF, or Markdown and save the report.
